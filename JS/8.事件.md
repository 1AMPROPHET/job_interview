## 事件绑定

```js
btn.addEventListener('click', e => {
  console.log('click')
})
// 通用事件绑定
function bindEvent(elem, type, fn) {
  elem.addEventListener(type, fn)
}
const a = document.getElementById('div')
bindEvent(a, 'click', e => {
  e.preventDefault() // 阻止默认行为
  alert('click')
})
```
