# 常用指令

* 登录
  * `ssh` username`@192.168.10.21` 回车
  * 密码
* 查看文件夹 `ls`
* 隐藏文件 `ls -a`
* 列表模式 `ll`
* 创建文件夹 `mkdir` docname
* 删除文件夹 `rm -rf` docname
* 访问目录 `cd`
* 修改文件名 `mv` oldname newname
* 移动文件夹 `mv` docname path
* 拷贝 `cp` doc cpdoc
* 新建文件 `touch` docname / `vi` docname 进入vim编辑器 esc : w q
